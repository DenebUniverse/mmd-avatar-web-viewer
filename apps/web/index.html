角色 / 架构

默认角色是绯英。启动服务前会扫描 assets/models，有效 PMX/PMD 会自动进入此列表；也可用 ?character=qianyeBlade 或 ?model=/assets/models/xxx/xxx.pmx 直达。

当前方案：PMX 模型 + VMD 动作 + VPD 姿态 + MMDAnimationHelper。资源优先使用 manifest 显示名，额外扫描到的有效格式也会动态加载。

表情 / 口型层

播放带 face/lip 的 VMD 时，建议保留 VMD 表情和口型；使用 TTS/麦克风驱动时再切回前端口型。

VMD 动作 / VPD 姿态

启动服务前会扫描 assets/motions 与 assets/poses。Registry 中有显示名的资源按 registry 展示，其他有效 VMD/VPD 按文件名展示。
