# 00 项目需求与实现情况

## 项目名称

```text
mmd-avatar-web-viewer
```

## 目标

做一个“米哈游直播间版”的 MMD 数字人二创原型。前端在本地浏览器中加载官方/用户提供的 PMX/PMD 角色模型，并通过标准 MMD VMD/VPD 资源播放动作、姿态、表情和口型。

当前运行架构：

```text
PMX / PMD 模型 + VMD 动作 + VPD 姿态 + MMDLoader + MMDAnimationHelper
```

## 当前功能

- 默认加载角色：绯英，路径为 `public/models/hiying_pmx/星穹铁道—绯英2.pmx`。
- 已接入角色：千冶·刃（含环），路径为 `public/models/qianye_blade_pmx/星穹铁道—千冶·刃2.pmx`。
- 前端角色列表：控制台可选择 manifest 中声明或自动扫描到的 PMX/PMD。
- 启动服务前自动扫描：`public/models`、`public/motions`、`public/poses`。
- 自动生成资源索引：`public/resource_manifest.json`。
- manifest 优先显示：`avatar_manifest.json` 与 `vmd_manifest.json` 中有 label/source 的资源按 manifest 展示；额外扫描到的合法资源按文件名展示。
- URL 直达角色：`?character=hiying`、`?character=qianyeBlade`，也可用 `?model=/models/local/xxx.pmx`。
- 加载 VMD 动作：`MMDLoader.loadAnimation()`。
- 驱动动作：`MMDAnimationHelper.add()` + `MMDAnimationHelper.update()`。
- 加载 VPD 姿态：`MMDLoader.loadVPD()` + `MMDAnimationHelper.pose()`。
- 前端控制面板：角色、VMD 动作、VPD 姿态均支持下拉列表；常用动作保留快捷按钮。
- 表情层：前端按钮或 VMD 自带 face 二选一。
- 口型层：前端 TTS/麦克风或 VMD 自带 lip 二选一。
- OBS 模式：`?obs=1`。

## 已接入模型

```text
public/models/hiying_pmx/星穹铁道—绯英2.pmx
public/models/qianye_blade_pmx/星穹铁道—千冶·刃2.pmx
public/models/qianye_blade_pmx/剑.pmx
public/models/qianye_blade_pmx/環.pmx
public/models/avatar_manifest.json
```

说明：前端角色选择默认加载主 PMX。`剑.pmx` 和 `環.pmx` 已作为附件资源保留，且被 manifest 标注为附件，不会误加入角色列表。
